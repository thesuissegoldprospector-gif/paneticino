# **App Name**: PaneDelivery

## Core Features:

- Home Feed: Display sponsored products and featured bakeries.
- Bakery Listings: Show a complete list of bakeries.
- Bakery Profile: Display bakery details, products, info, and ordering options.
- Geolocation Search: Search for bakeries that deliver to a specific address or postal code.
- Delivery zone check tool: The tool analyzes delivery information of local bakeries in order to identify valid delivery zones according to user input of comune/CAP.
- Customer Profile: Allow customers to manage personal data, addresses, order history, and favorite bakeries.
- Baker Profile/Dashboard: Enable bakers to manage company data, profile pictures, cover photos, products, orders, and delivery zones.

## Style Guidelines:

- Primary color: Golden yellow (#FFC107), evoking warmth and the color of baked goods.
- Background color: Pale yellow (#FAF9F0), creating a soft and inviting feel.
- Accent color: Light orange (#FFB74D), used for interactive elements and calls to action, providing contrast.
- Headline font: 'Playfair', serif, for elegance, especially on the Home; body font: 'PT Sans', sans-serif, for clarity.
- Use a bottom header navigation in the style of YouTube Shorts with sections for Home, Bakeries, Bakeries Near You, and Profile.
- Icons should be simple and related to bakery items and location.
- Subtle transitions and loading animations for a smooth user experience.
